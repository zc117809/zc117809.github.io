1.1 接收用户的输入

在python3：input会将用户输入的所有的内容都存成字符串类型

在python2中：
raw_input:用法与python3中的input一模一样
input：要求用户必须输入一个明确的数据类型，输入的是什么类型，就存成什么类型，很不方便用户的使用
age=input(“18”) age,type(age)
结果：18 (18, <type ‘int’>)
